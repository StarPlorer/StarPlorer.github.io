로봇 암 제어

  1. 목표 : 두산 로봇 암을 제어하는 방법을 학습
  2. 프로젝트 방향 : 물건 적재, 팔렛 타이징, 나사 조립

두산 로보틱스의 H-2515를 이용하여 여러 작업을 수행. 두산 로봇 티치 팬던트, 스튜디오, ROS2 총 3가지 제어 방법을 사용했음.
어떤 산업 현장에서 로봇 암이 무슨 목적으로 어떻게 사용되는지 고려하여 작업을 진행. 최대한 유연하게 코드작성

두산 로보틱스의 H-2515를 이용하여 여러 작업들을 제어했습니다. 산업 현장에서 쓰일 목적을 고려하여 적재, 팔렛타이징, 나사조립 프로젝트를 진행했습니다.
목표하는 작업을 진행하는 코드를 작성하는 것은 큰 어려움이 없었습니다. 중요한 것은 이 로봇 암이 어떤 환경에서 작동하는지 어떤 상황이 발생할 수 있는지 여러 시나리오를 고려하는 것이였습니다.
대표적인 시나리오는 전력 공급중단으로 인한 정지, 사람이 로봇 암에 끼인 경우를 고려하였습니다.
정지하고 재시작하는 시나리오를 클라이언트의 입장에서 만들었고 사람이 로봇 암에 끼인 경우를 대비해 민감도를 적절하게 조절하며 작업했습니다.
코드를 작성할 때 최대한 파라미터를 받아서 작동하도록 제작했습니다. 파라미터 기반으로 작동해야 다양한 환경에서 유연하게 작동할 수 있기때문입니다.
간단한 로봇 암 제어 동영상입니다.





나사 조립 프로세스를 작성했습니다. 나사가 있는 위 포지션에서 집게를 내리다가 적절한 충돌이 감지되면 나사로 간주하고 집게를 열어 나사를 잡습니다.
나사를 조립할 위 포지션에서 집게를 내리다가 적절한 충돌이 감지되면 나사를 회전시키며 내려갑니다. 첫 번째 영상은 나사를 충분히 돌리지못한채 내려가서 조립되지않았고
두 번째 동영상에서 개선해 나사를 조힙했습니다.





물건 적재, 팔렛타이징 프로세스입니다. 매우 섬세한 티칭이 필요했습니다. 허용 오차의 포지션에서 벗어나면 나중에 엉뚱한 위치에서 물건을 집거나 놓으려고합니다.
각 위치의 포지션을 가시화하기 위해 수성펜으로 표시하고 자를 이용해 티칭 값을 계산했습니다. 더 세밀하게 티칭하기 위해 반복 작업이 많은 구간이였습니다.





로봇 암을 이용해 컵 쌓기 시스템을 만들었습니다. 코드를 단순하고 유연성있게 작성했습니다.
위치 기반으로 물건을 집어 놓는 것보다 감지 기반으로 물건을 잡아 위치에 놓고 다음에 두어야할 위치를 계산하는 것이 더 유연했습니다.
