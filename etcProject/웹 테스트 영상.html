2021년 웹 테스트 영상

2021년에 웹 개발자로 취업 준비할 때 만들어본 웹 테스트 영상을 발견해서 업로드합니다. java,js,sql,SpringFrameWrok, MariaDB를 이용했던걸로 기억합니다. 옛 생각이 나서 반갑네요 ㅎㅎ
이후 웹 개발자로 일을 하며 웹 제작 기술을 향상시켰습니다. 웹은 모니터링, 관제 플랫폼이라고 생각합니다. 웹 제작 기술은 익혀두면 다양하게 사용가능합니다.
