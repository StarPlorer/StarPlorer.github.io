객체 인식

  1. 목표 : 모델을 학습시켜 객체 인식
  2. 프로젝트 방향 : 불량품 감지

데이터 라벨링 플랫폼을 이용하여 데이터를 만들고 YOLO 모델을 학습시킴. 라벨링 가이드를 모든 팀원이 동일하게 설정하는 것이 중요.

데이터 학습 플랫폼 슈퍼브AI를 사용해 데이터를 라벨링하고 모델에 데이터를 학습시켜 불량품을 예측했습니다.
예측률을 올릴방법을 고민하고 시도했습니다. 여러 테스트 결과 다음 방법들이 예측률 향상에 도움을 주었습니다.
1. 팀원들의 라벨링 가이드를 통일하기. 2. 데이터 더 많이 넣기. 3. 불량품을 검사하는 환경 통제하기
팀원들의 라벨링 기준이 생각보다 다양했습니다. 라벨링 가이드를 통일했습니다. 데이터를 더 많이 넣으면 당연히 모델의 성능이 향상될 것이라 생각하고 데이터를 더 넣었습니다.
검사 환경을 통제하였습니다. 환경을 제한하면 제한된 환경만 고려하면 되기에 예측률이 더 올라갔습니다. 위 세가지 방법을 적용한 결과 예측률이 약 30% 증가했습니다.
